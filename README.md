# ♟️ Sistema de Jogo de Xadrez em Java

Projeto desenvolvido durante o curso de Java do professor Nelio Alves,
com o objetivo de aplicar conceitos de Programação Orientada a Objetos
(POO) na construção de um jogo completo de xadrez executado no console.

O projeto implementa as regras fundamentais do jogo, movimentação das
peças, controle de turnos e validação de jogadas.

------------------------------------------------------------------------

# 📚 Conceitos aplicados

-   Programação Orientada a Objetos (POO)
-   Encapsulamento
-   Herança
-   Polimorfismo
-   Enumerações (Enum)
-   Tratamento de exceções
-   Estrutura de dados com matrizes
-   Organização em pacotes

# ♟️ Funcionalidades

-   Movimentação das peças
-   Validação de jogadas
-   Alternância de turnos entre jogadores
-   Verificação de xeque
-   Verificação de xeque‑mate

Movimentos especiais:

-   Roque
-   En Passant
-   Promoção de peão

------------------------------------------------------------------------

# ▶️ Como executar o projeto

### 1. Clonar o repositório

    git clone https://github.com/seuusuario/chess-system-java.git

### 2. Entrar na pasta do projeto

    cd chess-system-java

### 3. Executar o programa

Compile e execute:

    Program.java

------------------------------------------------------------------------

# 🎮 Como jogar

O jogo é executado no terminal.

Exemplo de jogada:

    e2
    e4

Primeiro selecione a peça e depois a posição de destino.
